How to use

- The standard size of the board is 16 x 16, if you would like to choose a different size, click on Size then select a value from 2 to 100 and the drawing board will automatically adjust to the size. (ie. if you choose 50 the board will become 50 x 50).

- To draw on the board simply hold the left mouse button down while dragging the mouse pointer over the boxes that you would like to color in.

- You have 2 options as far as how you would like to draw. You can select the color you want by pressing the color button. The color button will change to the color that you currently have selected, or you can choose rainbow which will give you a random color for every box.
